ExtJs快速入门之搭建Extjs开发环境(1)

发表于 2017-03-01   |   分类于 ExtJs   |  

下载ExtJs

目前ExtJs正式发行版本为6.2，由于访问Sencha的官网非常之卡，所以建议同学们选择国内的第三方下载。

首先，访问Sencha中文站，接着找到ExtJs6.2正式版下载之。在我写这篇文章的时候，这个地址仍然可用：http://pan.baidu.com/s/1hssLw5e

下载之后解压，我们发下整个框架的重量大概有574M，而我们开发时需要使用到的只有其中的很小一部分。

搭建项目骨架

我们打开解压后的ExtJs，大概构成为这样：

  • ext-bootstrap.js
  • build
  • packages
  • ……

我们这里大概目前只需要用到这几个。

打开我们的IDE，新建一个项目。我们大概需要这几个文件就可以搭建一个ExtJs骨架。

  1. ext-bootstrap.js
  2. build/ext-all.js
  3. build/ext-all-debug.js
  4. build/classic/locale/locale-zh_CN.js

大概如下图所示：

然后我们我们新建一个html和js文件来测试ExtJs是否加载后可用。

  • 新建一个app.js 

    Ext.onReady(function () {
    Ext.create('Ext.panel.Panel', {
    title: 'Hello',
    width: '100%',
    height: '100%',
    html: '<p>World!</p>',
    items:[{
        xtype : 'button',
        text : 'ClickMe',
        border : true,
        style : 'border : 1px blue solid',
        width : '10%'，
        handler : function(){
            alert('clicked');
        }
    }],
    renderTo: Ext.getBody()
    });
});

上面的代码我们创建了一个Panel，然后定义了一系列的配置。这个Panel下有个button，点击该button会调用'handler'所定义的function.
关于Extjs组件的各种定义和创建方式会在之后谈到，目前我们先把Extjs跑起来.

  • 新建一个index.html然后将以上说到的js文件按如下顺序引入： 

    <head>
<meta charset="UTF-8">
<title>ExtJs入门</title>
<script src="lib/extjs/ext-all.js"></script>
<script src="lib/extjs/ext-all-debug.js"></script>
<script src="lib/extjs/ext-bootstrap.js"></script>
<script src="lib/extjs/locale-zh_CN.js"></script>
<script src="app.js"></script>
</head>

  • 我们用浏览器打开这个index.html可以看到下图效果：

至此，页面使用了Extjs显示出了我们所定义的组件。

之后我们会在此基础上用Extjs逐渐构建自己的门户应用，通过完整的项目实践，认真地把各种坑填一发。
